W czym i jak został napisany GNU/linux


linux został głównie napisany w C (ok 99%) oraz assembly (0.9%)
jedyne pliki C++ w repo są tylko w tools/perf/util/c++ link

C++ nie został użyty jako główny język ponieważ jego zdaniem ten język jest okropnym językiem

mówi on że nieważne co C++ nie jest najbardziej poprawnym wyborem
C może być bezpośrednio użyty w programowaniu systemowym,
Najlepszym wyborem jest wybrać język z garbage colectorem (Odśmiecanie pamięci więcej o tym: info )

W programowaniu, tworzeniu własnego systemu najważniejszym wyborem zawsze będzie język programowania

jeżeli chcemy np pisać język za pomocą pythona (tak dla przykładu) to musielibyśmy napisać interpreter tego języka, Ale za to jeżeli wybierzemy język taki jak C to możemy zbudować system za pomocą kompilatora GCC (trzeba niestety zbudować lub pobrać z mirrora wersję i686) bez pisania własnego interpretera. Dalej będziemy musieli napisać własne biblioteki do obsługiwania np: VGA, TTY, GDT itp.
