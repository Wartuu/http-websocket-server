Na co nam pozwala GNU GPL


Licencja GNU GPL (3.0) pozwala nam na:

- Użytek komercyjny
- Modyfikację
- Dystrybucję
- Użycie patentowe (inspiracja, pisanie podobnej aplikacji)
- Użytkowanie prywatne

Pod warunkiem że spełnimy dane wymagania:

- Wzmianka o prawie autorskim i licencji
- Wzkazać zmiany jeżeli jest to program remixowanym (czyt. modyfikowany)
- W pełni otwarty kod źródłowy, dostępny dla każdego
- Program musi używać tej samej licencji (GNU GPL)

Licencja nie zapewnia użytkownikowi:

- Gwarancji działania aplikacji
- Autor nie bierze zrzeka się odpowiedzialności za możliwie wyrządzone szkody
